# 🥗 NutriAssist AI  
### Advancing Nutrition Science through Gemini AI  

An AI-powered web application that leverages **Google Gemini 2.5 Flash** to provide dynamic nutritional insights, personalized meal planning, and virtual nutrition coaching through an interactive Streamlit interface.

---

## 📌 Project Overview

NutriAssist AI is a smart nutrition assistant designed to help users:

- Understand detailed macronutrient and micronutrient composition of foods  
- Generate personalized meal plans based on health conditions and preferences  
- Receive AI-powered virtual nutrition coaching  

The application demonstrates the practical use of **Generative AI in healthcare and nutrition**, making dietary awareness accessible, interactive, and intelligent.

---

## 🎯 Objectives

- Develop an AI-powered nutrition analysis system  
- Provide macronutrient & micronutrient breakdown  
- Generate personalized meal plans  
- Offer AI-driven virtual coaching  
- Create a clean and user-friendly Streamlit interface  
- Enable real-time image + text-based nutrition analysis  

---

## ✨ Features

- 📸 **Dynamic Nutritional Insights (Text & Image Based)**  
- 🍽️ **Tailored Meal Planning**  
- 💬 **Virtual Nutrition Coaching**  
- 🤖 **AI-Powered Analysis using Gemini 2.5 Flash**  
- 📊 **Detailed Macro & Micro Nutrient Reports**  
- ⚡ **Fast Real-Time Response**  
- 🖥️ **Clean Streamlit Web Interface**

---

## 🧠 AI Model Used

**Gemini 2.5 Flash (`models/gemini-2.5-flash`)**

Selected because:

- Fast inference speed  
- Efficient resource usage  
- Strong contextual understanding  
- Real-time web integration support  
- Supports both text and image inputs
